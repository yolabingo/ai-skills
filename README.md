# ai-skills

Curated AI coding skills for Claude Code.

## What is this?

A public collection of reusable AI skills distributed as a Claude Code plugin marketplace. Each skill is a markdown-based plugin that enhances Claude Code with specialized capabilities.

## Install

Add the marketplace:

```bash
claude plugin marketplace add yolabingo/ai-skills
```

Install a skill:

```bash
claude plugin install <skill-name>
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [gh-intercept](skills/gh-intercept) | Shallow-clone GitHub repos in the background on fetch, serve subsequent searches from local cache |
